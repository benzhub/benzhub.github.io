CLI Shortcuts | Linux

掌握 Linux 命令行界面（CLI）可以提高效率和靈活性。我們將介紹一系列基本的 Linux CLI Shortcuts 方式。

主要 Linux CLI Shortcuts 方式:

  • Tab自動完成: 通過按Tab鍵快速完成命令、文件路徑或目錄名稱。
  • Ctrl + C: 使用此鍵盤快捷鍵即可立即停止運行中的進程或命令。
  • Ctrl + Z: 暫停進程，允許稍後恢復或終止。
  • Ctrl + D: 表示輸入的結束，通常用於關閉終端會話。
  • Ctrl + R: 通過命令歷史搜索，輕鬆找到並重新運行特定命令。
  • Ctrl + L: 清空終端屏幕，創建更整潔的工作空間。
  • Ctrl + A: 讓光標到最一開頭
  • Ctrl + E: 讓光標到最後
  • Ctrl + K: 刪除光標之後的所有東西刪除
  • Ctrl + U: 把整行command line 刪除
  • Ctrl + Y: 把Ctrl+K或Ctrl+U刪除的東西還原回去
    • Y ⇒ yank，在windows的剪貼版出現以前，在命令提示字元的時代都是用yank

演示命令行:

# Tab自動完成
ls /ho[TAB]user  # 自動完成為 ls /home/user

# Ctrl + C
ping example.com  # 按Ctrl + C停止ping進程

# Ctrl + Z
nano  # 啟動nano文本編輯器，然後按Ctrl + Z暫停它

# Ctrl + D
exit  # 使用Ctrl + D關閉終端會話

# Ctrl + R
# 按Ctrl + R，開始輸入命令，然後按Enter執行先前的命令

# Ctrl + L
# 清空終端屏幕以提高可見性

Linux CLI Shortcuts 優勢:

使用這些快捷方式可以讓Linux CLI Shortcuts快速導航、管理進程並優化工作流程，提升他們的命令行的效率。
