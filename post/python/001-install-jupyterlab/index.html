安裝&啟動JupyterLab | Python

2023/12/17

python |

Python JupyterLab 是一個功能強大的開源基於 Web 的 Python 交互式計算環境。它允許用戶創建並共享包含實時代碼、方程式、可視化和敘述文本的文檔。由於其多功能性和易用性， JupyterLab 支持各種語言並在數據科學、機器學習和科學計算中被廣泛使用。

# 安裝
python -m pip install jupyterlab

# 運行 JupyterLab
jupyter lab --ip=127.0.0.1 --port 8080

# 然後根據輸出的提示，使用瀏覽器連線 http://伺服器地址:8080 ，並使用輸出中提供的 token 直接登錄（或設置密碼後登錄）即可
