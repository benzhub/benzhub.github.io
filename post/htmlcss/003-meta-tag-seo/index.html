Meta Tag & SEO(meta標籤和搜尋引擎) | HTML

2023/12/20

html | seo |

學習撰寫描述、標題和其他，以提升你網站的可見性。無論你是網站所有者、開發人員還是 SEO 實踐者，了解這些 HTML 標籤策略對於優化你的網站存在、有效地觸及目標受眾至關重要。

HTML 的標籤和 SEO 設定對於提升你網站的排名有很大的影響！正確使用標籤可以顯著影響你的網站在搜尋引擎中的可見度。這篇部落格文章深入探討 HTML 標籤的基礎知識，探討它們如何設定 SEO

範例

<!DOCTYPE html>
<html lang="en">
  <head>
    <!-- 聲明文件使用的字符編碼 -->
    <meta charset="UTF-8" />
    <!-- 瀏覽器的兼容設定 -->
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <!-- 響應式設計的設定 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <!-- 搜尋引擎的內容與查找(通常不超過150個字) -->
    <meta name="description" content="這是我的第一個響應式頁面" />
    <!-- 搜尋引擎的關鍵字設定 -->
    <meta name="keywords" content="" />
    <!-- 禁止搜尋引擎機器人索引或者抓取信息 -->
    <meta name="robots" content="NOINDEX, NOFOLLOW" />
    <title>meta標籤</title>
  </head>
  <body></body>
</html>
