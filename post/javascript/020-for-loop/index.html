For Loop(迴圈) | Javascript

2023/12/30 2026/05/16
javascript

在 JavaScript 中，‘for’循環是一個強大的工具，允許開發人員遍歷一系列元素或執行一段程式碼特定次數。掌握’for’循環是 JavaScript 開發人員的基本技能。

See the Pen 020-For-Loop(迴圈)-Javascript by lenrich (@lenrich) on CodePen.

For循環的基礎:

for 循環的語法包括三個部分：初始化、條件和迭代表達式。
在這個例子中，循環將變量’i’初始化為0，只要’i’小於5就執行循環體，並在每次迭代後增加’i’。

"use strict";

for (let i = 0; i < 5; i++) {
  console.log(`迭代 ${i + 1}`);
}

遍歷陣列(Array):

for 循環通常用於遍歷陣列，按順序訪問每個元素。
這個循環打印 fruits 陣列的每個元素：

"use strict";

const fruits = ['蘋果', '香蕉', '橘子'];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

For循環的優勢:

  1. 精確性: 精確指定迭代次數。
  2. 陣列處理: 高效處理陣列並對每個元素執行任務。
