Linux or Mac 安裝 nvm(Node.js 的版本管理器) | Javascript

2024/01/01 2026/05/16
javascript nvm tools linux

管理 Node.js 版本對很多人來說是一場惡夢，但使用 nvm （Node.js版本管理器），這個過程變得簡單而高效。我們將介紹如何在Linux和Mac OS 系統安裝 nvm ，使他們能夠輕鬆切換 Node.js 版本。

1.下載nvm:

在終端中使用以下命令下載nvm。

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash

# or

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash

2.加載nvm:

  1. 安裝過程也會自動將 nvm 設定新增到你的設定檔中。
  • 如果你使用 zsh，則為 ~/.zshrc
  • 如果你使用的是 bash，則為 ~/.bash_profile
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"  # This loads nvm
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"  # This loads nvm bash_completion
  1. 加載nvm:
  • 關閉並重新打開終端，或運行以下命令來加載nvm。
source ~/.bashrc   # 對於Bash
source ~/.zshrc    # 對於Zsh
  1. 驗證安裝: 通過檢查其版本確認nvm是否正確安裝。
nvm --version

常用nvm 指令

# nvm 安裝最新的 npm 版本
nvm install-latest-npm

# 顯示nvm可以使用Node.js版本
nvm ls-remote

# 使用nvm安裝Node.js版本
nvm install v18.16.0

# 顯示所有已安裝的Node.js版本
nvm ls

# 切換到另一個Node.js版本
nvm use v16.19.1
