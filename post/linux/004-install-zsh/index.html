给Linux配置漂亮且强大的zsh | Linux

2023/12/16

linux |

Zsh 因為有自動補全指令的功能 ，因此深受工程師的喜愛，這邊我們使用熱門的oh-my-zsh，一步一步來安裝設定我們的 Zsh 吧!

# 安裝zsh
sudo apt install zsh

步驟

1. 安裝zsh

sudo apt install zsh

2. 安裝oh-my-zsh

oh-my-zsh 官方
  1. install
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
  1. 設定為默認的shell，這樣我們之後都會是使用zsh作為我們默認的zsh

3. 設定zsh好用的外掛(提示語外掛)

提示語外掛 官方

# 我們到oh my zsh的plugins的目錄裡面
cd ~/.oh-my-zsh/plugins/

# 我們直接clone下來我們要的外掛
git clone https://github.com/zsh-users/zsh-autosuggestions.git

# 我們在我們的設定檔中添加外掛的名稱
sudo vim ~/.zshrc

##############################
# plugins=(
#   git
#   zsh-autosuggestions
#)
##############################

# 刷新我們的zshrc
source ~/.zshrc
# 這樣我們就會看到指令的提示語了
