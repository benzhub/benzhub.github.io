在 Windows、Mac 和 Linux 上安裝 Git | Git

2024/01/18
git

Git 是一個強大的版本控制工具，用於跟蹤和管理項目的代碼。無論你是使用 Windows、Mac 還是 Linux，這個篇文章將為你提供在各個操作系統上安裝 Git 的詳細步驟。

1. Windows 安裝 Git

在 Windows 上安裝 Git 有多種方式，其中一種是使用 Git 官方發布的安裝程序。

步驟：

  1. 下載 Git 安裝程序（點擊這裡）
  2. 開始安裝程序，按照提示進行操作，選擇你需要的安裝選項。
  3. 在 “Choosing the default editor used by Git” 步驟中，建議選擇 “Use the Nano editor by default”，這是一個輕量且易於使用的文本編輯器。
  4. 在 “Adjusting your PATH environment” 步驟中，選擇 “Use Git from the Windows Command Prompt”。 點擊 “Next” 完成安裝。

驗證安裝：

在 PowerShell 中運行以下命令檢查是否安裝成功：

git --version

2. Mac 安裝 Git

在 Mac 上安裝 Git 通常使用 Homebrew 包管理器。

步驟：

  1. 打開終端（Terminal）。
  2. 安裝 Homebrew（如果未安裝）：
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安裝 Git：

brew install git

驗證安裝：

在終端中運行以下命令檢查是否安裝成功：

git --version

3. Linux 安裝 Git

在大多數 Linux 發行版上，你可以使用包管理器安裝 Git。

步驟：

  1. 在終端中運行以下命令（根據你的發行版選擇合適的命令）：
  • Ubuntu/Debian：
sudo apt-get update
sudo apt-get install git
  • Fedora：
sudo dnf install git
  • CentOS：
sudo yum install git

驗證安裝：

在終端中運行以下命令檢查是否安裝成功：

git --version

總結

無論你使用的是 Windows 、 Mac 還是 Linux ，安裝 Git 都是一個簡單的過程。這個指南提供了在各個操作系統上安裝 Git 的詳細步驟，以確保你可以順利開始使用這個強大的版本控制工具。
